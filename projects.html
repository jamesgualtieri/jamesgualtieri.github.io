 scotty3D

A full 3D graphics packaged application
 

Done as a part of my 15-462 class, the application wraps a mesh editor, a ray-tracing render, and an rigged animation suite.
 

The Mesh editor implements many simple operations such as edge flips, and surface bevels as well as more complex operations including Catmull-Clark Subdivisions and loop subdivision.
 

The renderer implements full ray-tracing to display realistic lighting and shadows. In addition, path tracing to render material properties such mirrors and glass surfaces is included using the Monte-Carlo method. In addition, there is support for environmental lighting maps.
 

Finally, the application includes a basic set of animation functions includinging basic spline interpolation, as well as pill capsule rigged vertices with skeleton inverse kinematics. In addition, there is a surface wave equation animation to create a ripple effect.

 phish

Final project for 15-112 class, a short themed game
 

Written in Python, the game is a simple top-down shooter that follows the avatar of a "Phishing" virus whose goal is to steal information from a user.
 

The visuals for the game are done entirely in Python's native GUI Tkinter library. Though not inherently a great graphics package, the game showcases the ability to create a visually interesting product with simple graphical tools.
